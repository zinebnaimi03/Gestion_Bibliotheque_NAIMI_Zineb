-- Projet Gestion Bibliothèque:

-Description:
Une application de bureau en Python pour gérer une bibliothèque : gestion des livres, des membres, emprunts/retours, et statistiques.
-Guide d'installation:
  avoir Python 3.x installé. 
  Cloner le dépôt ou télécharger les fichiers du projet.
  Installer les dépendances avec la commande :  
      pip install -r requirements.txt
  Pour utiliser l’application en mode console (terminal), lance 
      python src/main.py
  Pour utiliser l’application en mode interface graphique avec visualisations, lance :
      python src/visualisations.py
